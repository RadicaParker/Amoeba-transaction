# Radica Amoeba Internal Transaction

First Streamlit version structure for internal transaction submit, approval, record, and admin control.

## Features in this first structure
- Login
- Submit transaction
- Manual approver selection
- One-step approval queue
- Admin portal for users, amoebas, categories
- CSV and Excel export

## Default login
- Admin: `radicafinace` / `radica!23`
- Approver: `manager@radica.com` / `Admin123!`
- Submitter: `staff@radica.com` / `Admin123!`

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
